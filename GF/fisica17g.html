Rozamiento en un Plano Inclinado

El rozamiento en un plano inclinado es la fuerza que se opone al movimiento de un objeto sobre una superficie inclinada.

Cálculo del Rozamiento

  • La fuerza normal \( N \) se calcula como \( N = mg \cos \theta \).
  • El componente de la fuerza de gravedad paralelo al plano es \( F_{\parallel} = mg \sin \theta \).

Fórmulas
Fe ≤ μe N
Fc = μc N

Ejemplo 1: Objeto en Reposo

Calcular la fuerza de rozamiento estático para un objeto de 5 kg en un plano inclinado de 30° con un coeficiente de rozamiento estático de 0.4.


Datos: masa = 5 kg, \( \theta \) = 30°, μe = 0.4.
Operación Inicial Proceso Resultado
N = mg \cos \theta N = 5 kg × 9.8 m/s² × \cos 30° N ≈ 42.44 N
Fe ≤ μe N Fe ≤ 0.4 × 42.44 N Fe ≤ 16.98 N

Ejemplo 2: Objeto en Movimiento

Calcular la fuerza de rozamiento cinético para el mismo objeto si el coeficiente de rozamiento cinético es 0.2.


Datos: masa = 5 kg, \( \theta \) = 30°, μc = 0.2.
Operación Inicial Proceso Resultado
N = mg \cos \theta N = 5 kg × 9.8 m/s² × \cos 30° N ≈ 42.44 N
Fc = μc N Fc = 0.2 × 42.44 N Fc = 8.49 N
