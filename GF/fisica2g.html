Mediciones y Sistemas de Unidades

La medición consiste en determinar el valor numérico de una cantidad física.

¿Qué es una "Cantidad Física"?
Es todo lo que puede ser medido, por ejemplo: la longitud de una cuerda, el área o superficie de un terreno, el volumen de un cuerpo, la temperatura de una sustancia, el peso de una persona, etc.
Estas cantidades estan divididas en dos grupos
Cantidades Fundamentales Cantidades Físicas
Son un conjunto pequeño de cantidades que forman parte de casi cualquier elemento en el mundo. Se clasifican en unidades de medida, estas son una medida estándar o patrón que tiene un valor fijo.


¿Qué es un "Sistema de Unidades"?
Es un conjunto de unidades de medida consistente, normalizada y uniforme que nos permite obtener conversiones de un sistema a otro.
Los "Sistemas de Unidades" se clasifican en torno a sus unidades fundamentales

El desarrollo de la ciencia, el comercio y la cooperación internacional, ha llevado a la necesidad de contar con un sistema universal de unidades de medida.
Así en 1960 se adoptó, una forma revisada y complementada del sistema m.k.s para uso internacional; este sistema se conoce oficialmente como Sistema Internacional (SI).

longitud, masa, tiempo, corriente eléctrica, temperatura termodinámica, cantidad de sustancia e intensidad luminosa. Una vez determinadas estas cantidades definieron la unidad de medida o patrón de cada una de ellas.
